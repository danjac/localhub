{% trans "What is a Community?" %}

{% blocktrans %} A Community is a group of users. A Community can be based on an organization, hobby, physical location, affiliation or any other common interest. Think of a Subreddit or Facebook group, but one where the group is completely isolated from other groups - so anything you post on that group, stays safely inside that group. {% endblocktrans %}

{% trans "Starting a Community" %}

{% blocktrans %} If you are self-hosting please refer to the instructions on how to create a new community [LINK REQUIRED]. Otherwise you will need to contact the site admins of your LocalHub host to create a community for you. {% endblocktrans %}

{% trans "Joining a Community" %}

{% blocktrans %} There are two ways to join a Community (unless you have been assigned as a founding member by a site admin when setting up the Community). You can be sent an invite, or send a request to the Community Admins to join the Community. Details for Admins on managing invites and join requests are covered in the help page on Memberships [LINK REQUIRED]; this section is for prospective members wanting to join a Community. {% endblocktrans %}

{% trans "Join Requests" %}

{% blocktrans %} Once you have signed into Localhub, you can apply to join any public Community without needing an invite first. Just click on the link on the home page and confirm you wish to send a request. An email will be sent to the Community Admins [LINK REQUIRED]. Once they approve - or reject - your application, you should receive an email in response. If you are accepted, you will automatically become a member of the Community. {% endblocktrans %}

{% trans "Invites" %}

{% blocktrans %} An Admin [LINK REQUIRED] may send you an invite if they know your email address. You should receive an email with an invite link: if you click on the link you will automatically be added as a new member of the Community. Alternatively you may choose to reject the invite. {% endblocktrans %}

{% trans "Leaving a Community" %}

{% blocktrans %} To leave a Community permanently, first click on the My Settings link in the Sidebar, then select the Leave Community link in the settings Sidebar. Once you have left this Community you can only rejoin by sending a join request or accepting an invite. {% endblocktrans %}

{% blocktrans %} Your content - posts, photos, events as well as comments and private messages sent to other members - will not be visible to other members, but will not be deleted. You should delete any content before you leave, or request a Moderator [LINK REQUIRED] or Admin [LINK REQUIRED] do so. Remember that if you delete your account [LINK REQUIRED] completely, all your content, across all Communities, will be permanently deleted. {% endblocktrans %}

{% trans "Public and private Communities" %}

{% blocktrans %} Content published on public Communities is open to the world. Content published in private Communities is only available to members of that Community. Admins can change this setting in the Community Settings page. {% endblocktrans %}

{% trans "Local network" %}

{% blocktrans %} The Local Network lists all Communities available on this host. Public Communities are always visible. Private Communities are only shown if you are a member of these Communities. You can also search for a Community using the search field at the top of the page. {% endblocktrans %}

{% trans "Community settings" %} {% trans "Admins" %}

{% blocktrans %} The Community Settings link is located in the default Sidebar. The Community Settings page allows you to change a number of settings such as the name and logo of your Community, the description, terms and conditions, and some rules about membership and posting content. More detailed information on managing Community memberships can be found here [LINK REQUIRED]. {% endblocktrans %}

{% trans "Name, tagline and description" %}

{% blocktrans %} You can set the name of your Community, the tagline (which appears in the Local Network) and the full page description here. The description, if available, will be linked to at the bottom of the default Sidebar. You can use Markdown [LINK REQUIRED] in the description. The name of the Community will appear in the Local Network and the top left of the Navbar, unless replaced with a logo (see below). {% endblocktrans %}

{% trans "Community logo" %}

{% blocktrans %} You can upload a company logo, which will replace the name of the Community in the top left of the Navbar. The logo should be a PNG or JPEG file and will be resized to a max width/height of 100 pixels. An aspect ratio of 16/9 probably works best in terms of layout. {% endblocktrans %}

{% trans "Terms and conditions" %}

{% blocktrans %} The Terms field should include all terms and conditions you wish to apply to the Community. You can use Markdown [LINK REQUIRED] here. If you provide terms, a link will be added to the page in the default Sidebar. {% endblocktrans %}

{% trans "Public" %}

{% blocktrans %} Determines if the Community is public. Otherwise it is private. {% endblocktrans %}

{% trans "Content sensitive tags" %}

{% blocktrans %} By default, sensitive content is hidden from users [LINK REQUIRED]. This is managed in the Community Settings by determining which tags are considered "sensitive". For example, if you add #nsfw to the list of sensitive tags in the Settings, any post, event or photo with an #nsfw tag in the description will automatically be considered sensitive and the post will be hidden. {% endblocktrans %}

{% trans "Blacklisted emails" %}

{% blocktrans %} You can block join requests from individual email addresses or entire domains in the two fields Blacklisted email addresses and Blacklisted email domains respectively. Users with matching email addresses will not be able to send a request to join your community. For example, if you set gmail.com under Blacklisted email domains, a user with the email address "someone@gmail.com" will be blocked from sending a join request. {% endblocktrans %}
