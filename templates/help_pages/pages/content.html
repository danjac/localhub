{% trans "Posts, events and photos" %}

{% blocktrans %} LocalHub has three kinds of content: Posts, Events and Photos. In fact, the last two - Events and Photos - are just special kinds of Posts, and we refer to all three collectively as "Posts". All kinds of posts have a Description field, where you can add rich content using Markdown syntax. {% endblocktrans %}

  • {% blocktrans %} Posts are the most common form of content. A Post consists of a description or URL to an HTML page or image on the internet. You can optionally provide a title. You can create Posts if you want to link to an article on the web, display an external image e.g. from Imgur or Pinterest, embed a YouTube or Vimeo video, or write a post like you would on Twitter or Facebook. If you post a link to an HTML page and leave out the title, LocalHub will try and fetch the title of the page and insert this into the title for your post automatically. {% endblocktrans %}
  • {% blocktrans %} Events are posts with a specific future date and (optionally) a location. For example, you might want to let the Community know about a real-world or online meeting, or a concert, bake sale or meetup. If you provide a valid real-world location, LocalHub will display this location in an OpenStreetMap [LINK REQUIRED] map. {% endblocktrans %}
  • {% blocktrans %} Photos are uploaded photos and other images (JPEGs and PNGs). You can include some additional info about your photo such as information about the original artist or Creative Commons licensing. If you just want to show an image from a URL, however, you can just use a normal Post and either add the URL in the Link field or use Markdown syntax in the Description field to embed any number of images. {% endblocktrans %}
{% trans "Home page" %}

{% blocktrans %} All content - Posts, Events and Photos - are shown on the home page. By default, all content is shown. As a logged-in user, you can filter this content in your settings [LINK REQUIRED] or by blocking individual users [LINK REQUIRED] and tags [LINK REQUIRED]. If you are not logged in - assuming the Community is public - you will see all content. Content is always shown in order of most recent. {% endblocktrans %}

{% trans "Timeline" %}

{% blocktrans %} The Timeline provides a summarized, time-ordered view of all content. You can also filter by year and month. {% endblocktrans %}

{% trans "Adding content" %}

{% blocktrans %} You can publish a new Post, Event or Photo from the home page, or selecting Submit Post, Submit Event or Submit Photo in the Sidebar. Once you have published a post it will be immediately available to the rest of the Community. {% endblocktrans %}

{% trans "Resharing content" %}

{% blocktrans %} You can reshare posts published by other members by clicking the Reshare button. Resharing is useful if members are only following certain users, as they may not otherwise see this content - they will see content that is reshared by users they are following, even if they are not following the owner of the original post. {% endblocktrans %}

{% trans "#embedded-content" %}

{% blocktrans %} Embedded content here refers to links to certain sites which implement the Oembed protocol [LINK REQUIRED]. Examples include links to a YouTube or Vimeo video, Tweet, Facebook post, or Wordpress article. The content is typically embedded in an iframe. {% endblocktrans %}

{% blocktrans %} By default embedding is disabled if your browser enforces a "Do Not Track" (DNT) policy. If you wish to view embedded content, you can whitelist this site in your browser settings, or (as a logged in user) go to My Settings page and check the "Show embedded content" checkbox. {% endblocktrans %}

{% trans "Searching content" %}

{% blocktrans %} To find individual posts, just enter your search query in the box in the middle of the Navbar. You will be shown all posts, events and photos matching your query. {% endblocktrans %}

{% blocktrans %} You can also search specifically for Posts, Events and Photos in their individual pages. {% endblocktrans %}

{% trans "Liking content and Favorites" %}

{% blocktrans %} You can like any other members' content and comments. If you like a post or comment, the other user will receive a notification that you liked their content (although not if you "unlike" it). You can see all posts and comments you have liked in the Favorites page. {% endblocktrans %}

{% trans "Editing content" %}

{% blocktrans %} You can edit any of your own posts at any time, just by clicking the "Edit" button on the individual post. Moderators [LINK REQUIRED] can also edit other members' posts, for example if they want to add a content sensitive tag to the description. {% endblocktrans %}

{% trans "Deleting content" %}

{% blocktrans %} You can delete any of your posts at any time. Moderators [LINK REQUIRED] can also delete other members' posts. Note that if you delete a post, all comments belonging to that post - including those posted by other members - will also be deleted. If you delete your account, your posts (and subsequently comments) will be automatically deleted, across all Communities. If however you just leave a Community, your posts will not be visible, but will not be permanently deleted. Reshares will not themselves be deleted if the original post is deleted, but the content will be replaced by a note indicating that the original post no longer exists. {% endblocktrans %}

{% trans "Flagging content" %}

{% blocktrans %} You can "flag" content you object to if you consider it spam, offensive, in breach of the Community rules, and so on. You can flag any content or comment you do not own. Moderators [LINK REQUIRED] will be notified of any flagged content. Flagged posts will be highlighted to Moderators, and they will be able to see individual flags in the detail page of each post or comment. The Flags page, available only to Moderators, lists all flagged content across the site. It is up to Moderators how they wish to take action: whether to edit or remove content, escalate to have the offending member removed or suspended, or ignore the flag. {% endblocktrans %}
