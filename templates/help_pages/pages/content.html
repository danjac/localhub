{% trans "Posts, events and photos" %}

{% blocktrans %} LocalHub has three kinds of content: Posts, Events and Photos. In fact, the last two - Events and Photos - are just special kinds of Posts, and we refer to all three collectively as "Posts". All kinds of posts have a Description field, where you can add rich content using Markdown syntax. {% endblocktrans %}

  • {% blocktrans %} Posts are the most common form of content. A Post consists of a description or URL to an HTML page or image on the internet. You can optionally provide a title. You can create Posts if you want to link to an article on the web, display an external image e.g. from Imgur or Pinterest, embed a YouTube or Vimeo video, or write a post like you would on Twitter or Facebook. If you post a link to an HTML page and leave out the title, LocalHub will try and fetch the title of the page and insert this into the title for your post automatically. {% endblocktrans %}
  • {% blocktrans %} Events are posts with a specific future date and (optionally) a location. For example, you might want to let the Community know about a real-world or online meeting, or a concert, bake sale or meetup. If you provide a valid real-world location, LocalHub will display this location in an OpenStreetMap [LINK REQUIRED] map. {% endblocktrans %}
  • {% blocktrans %} Photos are uploaded photos and other images (JPEGs and PNGs). You can include some additional info about your photo such as information about the original artist or Creative Commons licensing. If you just want to show an image from a URL, however, you can just use a normal Post and either add the URL in the Link field or use Markdown syntax in the Description field to embed any number of images. {% endblocktrans %}
